Install & Use the Paraloom Plugin

Claude Desktop · ~5 minutes

Install

  1. Open Claude Desktop Settings
  2. Go to Plugins
  3. Add → Upload plugin
  4. Drop in paraloom-plugin.zip → Upload
  5. Connectors tab → Install by Paraloom
  6. Click Connect
  7. Sign in to Paraloom → shows Connected

Finish setup
